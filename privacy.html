개인정보처리방침

그때우리는 이용자의 개인정보를 소중하게 보호합니다.

그때우리는 주변의 추억 속 인연을 찾는 서비스입니다. 사용자의 정확한 위치는 다른 사용자에게 공개되지 않으며, 위치 공개는 사용자가 직접 선택한 경우에만 제한된 시간 동안 이루어집니다.

1. 수집하는 개인정보

  • 이메일 주소
  • 닉네임
  • 사용자 ID
  • 대략적인 위치 정보
  • 프로필 정보: 학교, 입학년도, 군 복무 부대, 회사 위치, 사는 곳, 고향 등 사용자가 입력한 정보
  • 채팅 요청, 채팅 메시지, 차단 및 신고 관련 정보
  • 푸시 알림 토큰
```

2. 개인정보 수집 목적

  • 회원가입 및 로그인
  • 주변 사용자 탐색 기능 제공
  • 학교, 군 복무 부대 등 공통 프로필 기반 매칭 기능 제공
  • 채팅 요청 및 채팅 서비스 제공
  • 차단, 신고 및 안전한 서비스 운영
  • 푸시 알림 제공
  • 서비스 오류 확인 및 품질 개선

3. 위치 정보 이용

그때우리는 주변 인연 찾기 기능을 제공하기 위해 위치 정보를 사용할 수 있습니다. 단, 위치 정보는 사용자가 앱에서 직접 위치 공개를 선택한 경우에만 사용됩니다.

회원가입 시 위치 공개 여부를 자동으로 설정하지 않으며, 사용자가 위치 공개를 원할 때마다 직접 위치 공개 버튼을 눌러야 합니다. 위치 공개는 제한된 시간 동안만 유지되며, 시간이 종료되면 자동으로 비공개 상태로 전환됩니다.

사용자의 정확한 위치는 다른 사용자에게 공개되지 않습니다. 서비스 내에서는 오차범위가 적용된 대략적인 위치만 표시되며, 주변 사용자 탐색과 매칭 기능을 위해서만 사용됩니다.

사용자는 언제든지 앱 설정 화면에서 위치 공개를 해제할 수 있습니다. 위치 공개를 해제하면 지도와 주변 사람 목록에서 표시되지 않습니다.

4. 위치 공개 및 수동 체크인

그때우리는 자동 체크인 기능을 제공하지 않습니다. 사용자가 자신의 위치를 지도 또는 주변 사용자 목록에 표시하려면 매번 직접 위치 공개를 선택해야 합니다.

위치 공개는 사용자가 선택한 제한 시간 동안만 유지되며, 해당 시간이 지나면 자동으로 종료됩니다. 앱을 다시 실행하더라도 사용자의 명시적인 선택 없이 위치 공개가 자동으로 다시 시작되지 않습니다.

5. 개인정보 보관 기간

개인정보는 회원이 서비스를 이용하는 동안 보관됩니다. 회원 탈퇴 또는 계정 삭제 시 프로필 정보, 위치 정보, 채팅 관련 정보 등 계정과 관련된 개인정보는 삭제됩니다. 단, 관계 법령에 따라 보관이 필요한 정보는 해당 기간 동안 보관될 수 있습니다.

6. 제3자 제공

그때우리는 이용자의 개인정보를 제3자에게 판매하거나 제공하지 않습니다. 단, 법령에 따른 요청이 있는 경우 예외적으로 제공될 수 있습니다.

7. 외부 서비스 이용

그때우리는 서비스 제공을 위해 Firebase Authentication, Cloud Firestore, Expo Push Notification 등 외부 서비스를 사용할 수 있습니다. 해당 서비스는 회원가입, 로그인, 데이터 저장, 푸시 알림 제공 등을 위해 사용됩니다.

8. 이용자의 권리

이용자는 언제든지 앱 내 계정 삭제 기능을 통해 개인정보 삭제를 요청할 수 있습니다. 또한 개인정보 열람, 수정, 삭제 및 처리 정지와 관련된 문의는 아래 이메일로 요청할 수 있습니다.

9. 아동 및 청소년 보호

그때우리는 위치 기반 주변 사용자 탐색 기능을 포함하고 있으므로 만 18세 이상 이용자를 대상으로 합니다. 만 18세 미만 이용자는 서비스를 이용할 수 없습니다.
```
